日本ひとり旅～Deep spot Japan～

静かな日本を映像と文章で伝える、YouTube連動型ポートフォリオサイト（SPA構成）

本サイトは、YouTubeチャンネル「異世界ひとり旅 ～Deep spot Japan～」の世界観をWeb上でも体験できるよう制作した、映像連動型のポートフォリオサイトです。
実際のYouTube動画をもとに、静けさと没入感を意識した構成・配色・タイポグラフィで設計。動画の魅力を文章でも丁寧に伝えることで、チャンネルのコンセプトである「知られざる日本」「静かなひとり旅」を表現しています。

制作のポイント

  • ajaxを使ったシームレスなページ遷移
  • HTML5のAPIを用い、各ページにユニークなURLとタイトルを最適化
  • 無限スクロール構成でもJavaScriptが無効な場合に対応できる形に調整
  • 各ページにトップページへ自動で戻るリダイレクト機能を実装
  • 相対パスではなくルートパスを使用し安定した遷移に対応
  • SPA(Single Page Application)の元となる考え方を理解する
使用言語
HTML CSS JavaScript jQuery
担当作業
HTMLマークアップ CSSコーディング JavaScript&jQueryの実装
使用プラグイン
clever-infinite-scroll

制作を通して学んだこと

jQueryを用いて、ページ遷移なしでコンテンツを読み込むSPAのような構成を実装しました。
ルーティング制御やレスポンシブ対応を通じて、UI設計や非同期処理の基礎を実践的に学ぶことができました。
また、映像と文章を組み合わせた情報設計の工夫を通して、ユーザー体験を意識したサイト制作の重要性を実感しました。
